Judul HTML

Judul HTML didefinisikan dengan tag < h1 > < head > < h6 >.
< h1 > Menentukan judul yang paling penting.
< h6 > Menentukan judul yang paling tidak penting.

Contoh Penulisan:



Judul Itu Penting

Mesin pencari menggunakan judul untuk mengindeks struktur dan konten halaman web Anda.
Pengguna sering kali membaca sekilas halaman berdasarkan judulnya. Penting untuk menggunakan judul untuk menunjukkan struktur dokumen.
< h1 >Judul harus digunakan untuk judul utama, diikuti oleh < h2 >judul-judul lain, kemudian judul yang kurang penting < h3 >, dan seterusnya.

  • < h1 > Judul Halaman
  • < h2 > Judul Bagian
  • < h3 > Sub-bagian
