Mengenal Dunia IT

Sebelum melangkah lebih jauh, yuk pahami dasar-dasarnya terlebih dahulu.

Teknik Informatika adalah disiplin ilmu yang mempelajari penerapan logika matematika dalam pengelolaan informasi menggunakan teknologi komputer. Jurusan ini berfokus pada proses pembuatan aplikasi, struktur data, algoritma, hingga perancangan arsitektur sistem.

Tidak harus pinter sih, tapi logika matematika sangat diperlukan. Mata kuliah seperti Matematika Diskrit atau Aljabar Linear akan sering ditemui karena sangat membantu dalam melatih algoritma dan logika berurutan saat menulis baris kode pemrograman.

Bahasa pemrograman adalah kumpulan instruksi atau sintaks yang digunakan untuk memberikan perintah kepada komputer agar melakukan tugas tertentu. Contohnya seperti HTML dan CSS untuk kerangka tampilan, JavaScript untuk interaktivitas web, atau Python dan C++.

Hardware adalah perangkat keras fisik yang bisa disentuh (seperti CPU, RAM, Monitor, Keyboard). Sedangkan Software adalah perangkat lunak atau program yang berjalan di dalam hardware tersebut (seperti Sistem Operasi Windows, Browser Chrome, atau game).

Database (basis data) adalah tempat berkumpulnya data-data yang tersimpan secara terstruktur di dalam server/komputer. Dengan database, informasi seperti data akun pengguna, nilai mahasiswa, atau stok barang dapat dikelola, diakses, dan diperbarui dengan cepat.

Peluangnya sangat luas di era digital. Lulusan jurusan ini banyak yang mengincar profesi strategis seperti Data Scientist, Web Developer, Software Engineer, System Analyst, UI/UX Designer, hingga ahli Cyber Security.
