Apa itu CSS??

Cascading Style Sheets (CSS) digunakan untuk memformat tata letak halaman web.

Dengan CSS, Kamu bisa mengontrol warna, font, ukuran teks, jarak antar elemen, bagaimana elemen diposisikan dan ditata, gambar latar belakang atau warna latar belakang apa yang akan digunakan, tampilan berbeda untuk berbagai perangkat dan ukuran layar, dan masih banyak lagi!



Sintaks CSS

Aturan CSS terdiri dari selektor dan blok deklarasi:

Selektor tersebut menunjuk ke elemen HTML yang ingin Anda beri gaya

Blok deklarasi berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma.

Setiap deklarasi mencakup nama properti CSS dan nilainya, dipisahkan oleh titik dua.

Beberapa deklarasi CSS dipisahkan dengan titik koma, dan blok deklarasi dikelilingi oleh kurung kurawal.

Contoh Penulisan:

p adalah selektor dalam CSS (menunjuk ke elemen HTML yang ingin Anda beri gaya: < p >).

color adalah sebuah properti, dan redadalah nilai properti tersebut.

text-align adalah sebuah properti, dan centeradalah nilai properti tersebut.
